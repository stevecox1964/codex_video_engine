# Tiny Robot Baker

## Goal

A 20 second animated story about a tiny robot baker learning teamwork in a cozy bakery.

## Video Type

animated_story

## Audience

General audience

## Format

- Aspect ratio: 16:9
- Target duration: 20 seconds
- Style: Warm cinematic 3D animation
- Voiceover: yes

## Constraints

- Must include: A clear beginning, middle, and ending
- Must avoid: Unreadable generated text, logos, watermarks

## Approval Points

- Approve scene plan before paid generation.
- Approve still frames before image-to-video.
- Approve voiceover script before TTS.
- Approve final clips before assembly.
