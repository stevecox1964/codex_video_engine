# Animated Website Story

## Goal

Create a short visual proof for a video idea.

## Video Type

product_promo

## Audience

Small business owners

## Format

- Aspect ratio: 16:9
- Target duration: 20 seconds
- Style: Cinematic, practical, modern
- Voiceover: yes

## Constraints

- Must include: Not specified.
- Must avoid: Unreadable generated text, logos, watermarks

## Approval Points

- Approve scene plan before paid generation.
- Approve still frames before image-to-video.
- Approve voiceover script before TTS.
- Approve final clips before assembly.
